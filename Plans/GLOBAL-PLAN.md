# 🧭 Глобальный план — Signal Dashboard Platform

## Видение

Построить **сигнальный дашборд**, куда токены попадают в разные колонки в зависимости от стратегии подбора. 
Каждая колонка = одна стратегия (например, кластерная стратегия = одна колонка).
Для каждой колонки рассчитывается средний % роста за день, чтобы знать как флипать.

В перспективе — стримить платформу, набрать аудиторию, продавать доступ за подписку.

---

## Фаза 1: Стратегии

Собрать и запрограммировать несколько стратегий для подбора прибыльных токенов:

- [ ] **Кластерная стратегия** — пересечения KOL/SM кошельков (текущая работа)
- [ ] **Narrative/Early meme** — виральные мемы, Twitter/X тренды
- [ ] **Dev tracking** — проверенные dev'ы с историей
- [ ] **Deep-dip flip** — свежие токены после -70%+ дампа
- [ ] **Spring Load** — консолидация → импульс
- [ ] **DEX Boost momentum** — органичный рост после буста
- [ ] *другие по мере нахождения*

Каждая стратегия должна выдавать:
- Токен (адрес + символ)
- Время сигнала
- Вес/скор сигнала
- Средний % роста за день (исторический по стратегии)

---

## Фаза 2: Сигнальный дашборд

Собрать веб-дашборд с колонками по стратегиям:

```
┌──────────┬──────────┬──────────┬──────────┐
│ Cluster  │ Early    │ Dev      │ Deep-Dip │
│ Strategy │ Meme     │ Tracking │ Flip     │
├──────────┼──────────┼──────────┼──────────┤
│ TOKEN A  │ TOKEN C  │ TOKEN E  │ TOKEN G  │
│ TOKEN B  │ TOKEN D  │ TOKEN F  │ TOKEN H  │
│          │          │          │ TOKEN I  │
├──────────┴──────────┴──────────┴──────────┤
│ Средний % роста за день по каждой колонке  │
└───────────────────────────────────────────┘
```

- Каждая колонка показывает токены, найденные этой стратегией
- Под каждой колонкой — средний % роста за день
- Возможность фильтрации по MC, времени, скору
