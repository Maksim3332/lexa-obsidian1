# Market Correlations

## Назначение
Причинно-следственные связи между рыночными условиями и вероятными исходами.
Только "если X, то вероятно Y". Не содержит стратегий и рекомендаций.

## Launch Activity ↔ Narrative Quality

| Уровень запусков | Эффект |
|------------------|--------|
| <50/день | Каждый токен получает больше внимания, выше шанс пампа |
| 50-200/день | Здоровый рынок, сильные нарративы выделяются |
| >200/день | Attention dilution, средняя выживаемость падает |
| >500/день | Перегрев, только Tier S нарративы выживают |

**Правило:** Чем выше launch velocity, тем сильнее должен быть нарратив для выживания.

## Migration Rate ↔ Survival Probability

| Migration Rate | Выживаемость |
|---------------|-------------|
| <1% | Экстремально низкое качество |
| 1-3% | Норма, ~30% migrated выживают >24ч |
| 3-5% | Сильный рынок, ~50% выживаемость |
| >5% | Эйфория, но много фейковых миграций |

**Паттерн:** Высокий migration rate + падающий launch count = улучшение качества рынка.
**Паттерн:** Высокий migration rate + растущий launch count = пузырь.

## Liquidity Regime ↔ Runner Potential

| Liquidity Regime | Max Runner MC | Кол-во runners | Риск |
|-----------------|---------------|----------------|------|
| Abundant | $10M+ | 5-10 | Low |
| Concentrated | $5-10M | 1-3 | Medium |
| Thinning | $2-5M | 0-1 | High |
| Drained | <$1M | 0 | Extreme |

**Правило:** В thinning liquidity только первый токен нового нарратива имеет шанс. Копии умирают.

## Trader Psychology ↔ Dip Recovery

| Режим | Recovery после −60% |
|-------|-------------------|
| Mania | <15 мин |
| Healthy | 30-60 мин |
| Cautious | Только топ-нарративы |
| Exhausted | Recovery нет, −80% = death |
| Despair | −50% = liquidation cascade |

**Mental Model:** The willingness to buy dips is the single strongest indicator of market health.
