# Market Regimes

## Назначение
Макро-анализ рынка Pump.fun. Только факты и классификация текущего состояния.
Не содержит стратегий, паттернов или рекомендаций — только "где мы сейчас".

## Классификация режимов

| Score | Режим | Описание |
|-------|-------|----------|
| 8-10 | Euphoria / Mania | Запуски зашкаливают, агрессивные покупки, экспоненциальный объём |
| 6-8 | Healthy Speculation | Умеренный рост, selective dip buying, здоровый объём |
| 4-6 | Cautious / Mixed | Объём падает, трейдеры выборочны, только сильные нарративы |
| 2-4 | Exhaustion / Decline | Мало запусков, минимальный объём, почти нет runners |
| 0-2 | Despair / Risk-Off | Нулевая активность, капитал ушёл |

## Режимы ликвидности

| Режим | Описание |
|-------|----------|
| Abundant | Объём растёт, multiple runners, liq быстро пополняется |
| Concentrated | Рынок фокусируется на 1-2 нарративах |
| Thinning | Объём падает, только сильнейшие выживают |
| Drained | Liq уходит с платформы, почти нет runners |
