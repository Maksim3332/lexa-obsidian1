# Top Holders Database

Холдеры, которые держали топ-позиции в токенах выросших на 50%+.

## Структура

| Файл | Описание |
|------|----------|
| `top_holders.json` | Мастер-база холдеров с токенами, долями, USD value |
| `README.md` | Этот файл |

## Как пополняется

1. Пост-анализ находит токен, выросший на 50%+
2. Извлекает топ-20 холдеров через `gmgn-cli token holders`
3. Записывает в базу: адрес, доля (amount_percentage), USD value, тэги

## Использование

- Если один холдер сидит в 3+ успешных токенах → сильный сигнал
- Если у холдера тэги `bundler`, `sniper`, `rat_trader` → оценивать риски
- Анализ концентрации: если топ-10 держат >50% → высокий риск дампа
